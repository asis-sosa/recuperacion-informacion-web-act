# **Hacks de Ecosia**

*Tip para Ecosia*

>Buscas como en Bing/Yahoo

>Pero ayudas al medio ambiente

>Ideal para investigación, datasets y documentación técnica

# *1.- Frase exacta*

Busca una frase exactamente como la escribes.

Ejemplo: "desarrollo web con react"

# *2.- Excluir -palabra*

Excluye resultados que contengan una palabra específica.

Ejemplo: javascript -angular

# *3.- OR*

Busca uno u otro término (en mayúsculas).

Ejemplo: vue OR react

# *4.- AND*

Obliga a que aparezcan ambos términos.

Ejemplo: docker AND kubernetes

# *5.- Site:*

Busca resultados dentro de un sitio específico.

Ejemplo: site:github.com tailwind

# *6.- -Site:*

Excluye un dominio completo.

Ejemplo: linux -site:stackoverflow.com

# *7.- Filetype:*

Busca archivos de un tipo concreto.

Ejemplo: filetype:pdf redes neuronales

# *8.- Intitle:*

El término debe aparecer en el título.

Ejemplo: intitle:tutorial laravel

# *9.- Allintitle:*

Todas las palabras deben estar en el título.

Ejemplo: allintitle:api rest node

# *10.- Inurl:*

El término debe aparecer en la URL.

Ejemplo: inurl:admin

# *11.- Allinurl:*

Todas las palabras deben aparecer en la URL.

Ejemplo: allinurl:login panel

# *12.- * (Comodín)*

Sustituye palabras desconocidas dentro de una frase.

Ejemplo: "cómo instalar * en ubuntu"

# *13.- () (Agrupación)*

Agrupa términos para búsquedas complejas.

Ejemplo: (frontend OR backend) AND developer

# *14.- Define:*

Obtiene definiciones rápidas.

Ejemplo: define:microservicios

# *15.- Related:*

Busca sitios web similares a otro.

Ejemplo: related:github.com

# *16.- Cache:*

Muestra la versión almacenada en caché de una página.

Ejemplo: cache:example.com

# *17.- Before:*

Resultados publicados antes de una fecha aproximada.

Ejemplo: blockchain before:2020

# *18.- After:*

Resultados posteriores a una fecha.

Ejemplo: astro after:2022

# *19.- Site:.edu / Site:.gov*

Búsqueda académica o gubernamental.

Ejemplo: site:.edu machine learning

# *20.- Filetype:csv OR filetype:xlsx*

Encuentra datasets y datos abiertos.

Ejemplo: filetype:csv population statistics
