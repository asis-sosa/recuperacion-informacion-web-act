# **Hacks de Baidu**

*Tips para Baidu*

>Prioriza chino simplificado → mejores resultados.

>Combina inglés + chino para temas técnicos.

>Muchos blogs técnicos chinos están en csdn.net, zhihu.com, juejin.cn.

# *1.- Frase Exacta ( 精确匹配 )*

Busca la frase exactamente como se escribe.

Ejemplo: "人工智能 机器学习"

# *2.- Excluir palabras (-palabra)*

Excluye resultados que contengan una palabra.

Ejemplo: 区块链 -比特币

# *3.- OR / |（或）*

Busca uno u otro término.

Ejemplo: Python | Java

# *4.- Site:（站点限定）*

Busca dentro de un sitio específico.

Ejemplo: site:github.com 深度学习

# *5.- -Site:（排除站点）*

Excluye un sitio completo.

Ejemplo: Docker -site:csdn.net

# *6.- Intitle:（标题包含）*

La palabra debe aparecer en el título.

Ejemplo: intitle:教程 Vue

# *7.- Allintitle:（标题完全匹配）*

Todas las palabras deben estar en el título.

Ejemplo: allintitle:Spring Boot 教程

# *8.- Inurl:（URL包含）*

Busca palabras dentro de la URL.

Ejemplo: inurl:login

# *9.- Filetype:（文件类型）*

Busca archivos por tipo.

Ejemplo: filetype:pdf 数据挖掘

# *10.- Rango 数字范围搜索（数字区间）*

Busca valores dentro de un rango.

Ejemplo: 手机 3000-5000 元

# *11.- Filtro 时间筛选（按时间）*

Filtra resultados por periodo desde las herramientas.

Ejemplo: 人工智能

*(luego usar filtro “时间” en Baidu)*

# *12.- Comodin ( * )（通配符）*

Reemplaza palabras desconocidas dentro de una frase.

Ejemplo: "如何 使用 * 框架"

# *13.- Parentesis ()（分组）*

Agrupa términos para búsquedas complejas.

Ejemplo: (React | Vue) 前端

# *14.- Define:（定义）*

Obtén definiciones rápidas.

Ejemplo: define:区块链

# *15.- Related:（相关网站）*

Encuentra sitios similares a otro.

Ejemplo: related:zhihu.com

# *16.- Cache:（网页缓存）*

Muestra la versión almacenada por Baidu.

Ejemplo: cache:example.com

# *17.- Site:.edu.cn（教育资源）*

Busca contenido académico chino.

Ejemplo: site:.edu.cn 人工智能

# *18.- Site:.gov.cn（政府资源）*

Busca información gubernamental oficial.

Ejemplo: site:.gov.cn 网络安全

# *19.- 关键词 + 文档类型*

Encuentra manuales y guías técnicas.
Ejemplo: Linux 运维 手册 filetype:pdf

# *20.- 中文 + 英文关键词混合搜索*

Baidu funciona mejor mezclando chino + inglés.

Ejemplo: 机器学习 machine learning tutorial
