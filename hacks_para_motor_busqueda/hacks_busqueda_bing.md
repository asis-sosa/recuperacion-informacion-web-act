# **Hacks de Bing**

# *1.- Más ( + )*

Obliga a incluir todos los términos precedidos por el símbolo +. Muy útil para asegurarse de que una palabra no se ignore en la búsqueda.

# *2.-"Frase Exacta"*

Colocar entre comillas dobles fuerza a buscar esa frase exacta, ideal para nombres propios, citas o términos compuestos.

# *3.- – o NO*

Elimina de los resultados cualquier página que contenga el término a continuación.

# *4.- O u OR*

Permite encontrar páginas que contengan cualquiera de los términos indicados. Puede usarse el símbolo | (barra vertical) como alternativa.

# *5.- Y o AND*

Asegura la presencia de todos los términos o frases.

# *6.- Paréntesis ()*

Agrupa términos o condiciones para establecer prioridades en búsquedas complejas.

# *7.- Site*

Busca solo dentro de un dominio o subdominio concreto, por ejemplo: site:microsoft.com inteligencia artificial.

# *8.- loc: o location*

Filtra los resultados por ubicación geográfica.

# *9.- Filetype*

Busca solo archivos de un tipo concreto: filetype:pdf, filetype:doc, etc.

# *10.- inbody:, inanchor:, intitle:, inurl*

Busca páginas donde la palabra clave aparezca en el cuerpo, anclaje, título o URL respectivamente.

# *11.- Define*

Ofrece la definición de una palabra.

# *12.- Domain*

Limita los resultados a un dominio específico de alto nivel.

# *13.- Contains*

Muestra páginas que contengan enlaces a archivos de un tipo especificado.

# *14.- Feed*

Encuentra páginas que contengan feeds RSS o Atom relacionados con la búsqueda.

# *15.- Language*

Restringe los resultados a un idioma determinado.

# *16.- IP*

Encuentra sitios asociados a una dirección IP concreta.

# *17.- Near*

Operador de proximidad; permite buscar términos que aparecen próximos entre sí, por ejemplo: ipad near:5 apple muestra páginas donde las palabras ‘ipad’ y ‘apple’ estén separadas por un máximo de cinco palabras.

# *18.- Keyword*

Marca una palabra clave específica dentro de la búsqueda.

# *19.- Encuentra información exacta*

Usa comillas dobles para buscar frases tal cual, cada vez que necesites datos exactos (títulos de libros, citas, números de serie…).

# *20.- Refina tus búsquedas temáticas*

Agrupa conceptos con paréntesis y operadores lógicos: (marketing digital OR posicionamiento) AND Bing.
