# **Hacks de Yahoo!**

*Tip extra*

*Yahoo usa Bing como backend, así que muchos hacks funcionan igual en ambos motores. Si dominas estos, prácticamente dominas dos buscadores al mismo tiempo*

# *1.- Frase Completa*

Busca una frase tal cual, sin variaciones.

Ejemplo: "desarrollo web con laravel"

# *2.- Excluir una palabra al usar guion ( -palabra)*

Excluye una palabra de los resultados.

Ejemplo: javascript -react

# *3.- OR*

Busca uno u otro término (en mayúsculas).

Ejemplo: vue OR react

# *4.- AND*

Fuerza que ambos términos aparezcan.

Ejemplo: laravel AND postgres

# *5.- Site*

Busca dentro de un sitio específico.

Ejemplo: site:github.com tailwind

# *6.- -Site*

Excluye un sitio completo.

Ejemplo: docker -site:medium.com

# *7.- Filetype*

Busca archivos de un tipo específico.

Ejemplo: filetype:pdf redes neuronales

# *8.- Intitle*

La palabra debe aparecer en el título.

Ejemplo: intitle:tutorial astro

# *9.- Allintitle*

Todas las palabras deben estar en el título.

Ejemplo: allintitle:api rest laravel

# *10.- Inurl*

La palabra debe aparecer en la URL.

Ejemplo: inurl:login

# *11.- Allinurl*

Todas las palabras deben estar en la URL.

Ejemplo: allinurl:admin panel

# *12.- * (comodín)*

Sustituye una o varias palabras desconocidas.

Ejemplo: "cómo instalar * en linux"

# *13.- () (Agrupación)*

Agrupa términos para búsquedas complejas.

Ejemplo: (docker OR kubernetes) AND backend

# *14.- Define*

Muestra definiciones rápidas.

Ejemplo: define:biometría

# *15.- Related*

Busca sitios relacionados con otro.

Ejemplo: related:stackoverflow.com

# *16.- Cache*

Muestra la versión en caché de una página.

Ejemplo: cache:example.com

# *17.- Before: / after*

Filtra resultados por fecha aproximada.

Ejemplo: astro after:2022

# *18.- "Palabra1 palabra2" -otra*

Frase exacta + exclusión.

Eejmplo: "machine learning" -python

# *19.- Site:.edu / site:.gov*

Busca contenido académico o gubernamental.

Ejemplo: site:.edu inteligencia artificial

# *20.- Filetype:xlsx OR filetype:csv*

Encuentra datos abiertos y tablas.

Ejemplo: filetype:csv estadisticas poblacion
